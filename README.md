# Array

Projeto simples em Java para praticar o uso de arrays.

## Descrição

O programa cria um array de caracteres com as letras `A`, `B` e `C`, percorre o array com um laço `for` e exibe os valores no terminal separados por hífen.

## Código principal

```text
array.java
```

## Como executar

```bash
javac array.java
java array
```

## Exemplo de saída

```text
A-B-C
```

## Tecnologias utilizadas

- Java

## Objetivo

Praticar declaração de arrays, acesso por índice, repetição com `for` e formatação simples de saída no console.
